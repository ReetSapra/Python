Python Projects

This repository contains a collection of Python projects demonstrating problem-solving, data processing, and interactive game development. Each project was created to strengthen programming fundamentals, logical thinking, and practical Python skills.

Projects
Birthday CSV Project

A program that reads and processes birthday information stored in a CSV file. Demonstrates file handling, data organization, and working with structured datasets.

Word Guessing Game

An interactive command-line game where users attempt to guess a hidden word. Demonstrates loops, conditionals, user input handling, and game logic.

Crack the Code

A logic-based guessing game in which players attempt to discover a secret code using clues and deduction. Demonstrates problem-solving algorithms and interactive program design.

Integer Set Analyzer

A program that analyzes sets of integers and performs various calculations and evaluations. Demonstrates data structures, mathematical operations, and analytical thinking.

Odd and Even

A simple program that identifies and processes odd and even numbers. Demonstrates conditional statements and numerical operations.

20-Side Dice Game

A simulation of a 20-sided dice game that incorporates randomness and user interaction. Demonstrates the use of Python's random module and game mechanics.
